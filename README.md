# ReBooks
### 프로젝트 소개

---

- 네이버 책 API를 사용하여 사용자가 리뷰하고 싶은 도서를 검색할 수 있다.
- 선택한 도서의 상세정보를 확인하고 한 줄 리뷰를 작성할 수 있다.
- 다른 회원의 리뷰를 확인 할 수 있다.
- 가볍고 설치가 쉬운 h2 Database를 사용했다.
- Java, Spring Boot를 사용했다.

### 기능

---

- 리뷰 할 도서 검색
- 리뷰 작성/등록/삭제
- 회원 가입 기능
- 로그인/로그아웃
- 전체 리뷰 조회
- 도서 제목으로 원하는 도서의 리뷰만 조회


### 사용 기술 및 라이브러리

---

- 네이버 검색 책 API 사용
- Java
- Spring Boot
- h2
